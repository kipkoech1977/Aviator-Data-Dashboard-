# ⚡ Quick Start - 5 Minutes to Install

## 🎯 For Infinix Users - Easiest Way!

### **NO COMPUTER NEEDED!**

---

## 📱 **On Your Infinix - 5 Simple Steps:**

### **1️⃣ Enable Unknown Sources**
```
Settings → Security → Install Unknown Apps → ON
```

### **2️⃣ Download APK**
```
Chrome → GitHub Releases → Download APK
```

### **3️⃣ Install**
```
File Manager → Downloads → Tap APK → Install
```

### **4️⃣ Grant Permissions**
```
App opens → Tap ALLOW for all permissions
```

### **5️⃣ Add to Home Screen**
```
Long-press icon → Add to Home Screen → Done!
```

---

## 🎉 **That's It! App is Ready to Use!**

Tap the icon on your home screen and enjoy! 🚀
